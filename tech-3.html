Color-Blocking

Color blocking is a system of creating a full range of colors using a set of primary colors.
There are two color models: additive and subtractive.
These are based on how colors are created.

A true primary color is a color that is not made up of any other colors than its own.
The secondary colors in the YRB system is created by mixing two primaries.
Orange = yellow + red Violet = red + blue Green = blue + yellow

The tertiary colors in the YRB system is created by mixing primary and secondary colors.
